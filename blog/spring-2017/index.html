Posted on April 20, 2017

Where’s my time going?

Hello readers (mom), I apologize for my loooong delay between post. I can hardly believe it is already spring time. Whew, time is traveling faster now. What have you been up to so far this year? I hope something awesome!

Then I’ve taken on building a desktop application for an engineering firm. On this project, I am on my own, so designer, manager, and developer! Focused on maintainability, I’m following test driven development (TDD) principles and loving it. These two project are definitely taking up the vast majority of my time, but I’m always trying to keep learning.

On the learning front, I’ve launched two open source projects (my rad Github). One is the theme to this blog, beautifultom, and the other is a tool for creating rainfall files that work with EPA SWMM, EPA SWMM Rainfall File Creator. I’m hoping to learn more about Git, while learning to manage an OS project, and working with other developers! I hope that you find either of these tools useful and will help me out making them better for everyone.
